# Python-for-Data-Science-Project
## Stock Price and Revenue Analysis: Tesla & GameStop

## Overview
This project analyzes the historical stock prices and quarterly revenue of Tesla and GameStop using Python. It demonstrates how to collect financial data from multiple sources, clean and prepare the data, and visualize the relationship between stock performance and company revenue.

## Objectives
* Extract historical stock data using the `yfinance` library.
* Scrape revenue data from web pages using `requests` and `BeautifulSoup`.
* Clean and transform the extracted datasets.
* Visualize stock prices and revenue using an interactive dashboard.

## Tools and Libraries
* Python
* Pandas
* yfinance
* Requests
* BeautifulSoup
* Plotly

## Project Workflow
1. Retrieved Tesla and GameStop stock data using the `yfinance` API.
2. Scraped quarterly revenue data from HTML pages.
3. Cleaned the revenue data by removing currency symbols, commas, and empty values.
4. Created data visualizations comparing stock prices and company revenue.

## Key Learning Outcomes
* Working with financial datasets.
* Web scraping using BeautifulSoup.
* Data cleaning and preprocessing with Pandas.
* Building simple financial dashboards and visualizations.

## Dataset Sources
